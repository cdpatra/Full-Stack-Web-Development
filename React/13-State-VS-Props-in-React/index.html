State VS Props in React

Difference between State and Props

State

  • State is a local variable of any particular component, it means state is controlled by the component
  • State is Mutable

Props

  • Props are controlled by the parent component
  • Props are Immutable (but in React we can change props but it is not good practice)

But the main similarity between states and props is that whenever either of them gets changed, our component get re-rendered

Data Types in JavaScript

There are two types of data types in JavaScript

  1. Primitive

  2. Non Primitive

We can classify primitive data types in 7 categories.

  1. number (in JS there is only one data types of all types of numbers, therefore there is nothing like float,int,double in JS)
  2. string
  3. boolean
  4. undefined
  5. null
  6. bigint
  7. symbol

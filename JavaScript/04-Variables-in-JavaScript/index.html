Variables in JavaScript

A JavaScript variable is jsut a name of storage location.

And we can give that name in three ways

  • Using let keyword
  • Using const keyword
  • Using var keyword

Rules for naming variables.

A variable name can contain ony for the the following characters but must not start with a digit.

  • Capital Letters: A to Z
  • Small Letters: a to z
  • Digits: 0 to 9 (but we can't start variable name with digits)
  • Only two special characters that are: Dollar "$" and Underscore "_".

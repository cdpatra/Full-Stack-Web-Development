Math in JavaScript

Mathematical Operators

  1. Addition Operator: +
  2. Subtraction Operator: -
  3. Multiplication Operator: *
  4. Division Operator: / (The quotient will be in decimal i.e. floating point)
  5. Remainder Operator: %
  6. Exponentiation Operator: **

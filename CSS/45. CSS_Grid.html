CSS Grid

  • CSS Grid is the new layout model in CSS
  • From october 2017 all major browser started supporting CSS Grid
    • Earlier we were using table to make layout but it is not very feature rich, then later we started using div and span containers, then we starder using CSS flexbox, and now wer are using CSS Grid

Both flexbox and CSS Grid has its own importance and should be used for designing a web page.

  • Grid workds for 2 dimensional alignment
  • Flexbox is good for space distribution

Terminologies of CSS Grid

1. Grid Columns

The vertical lines of gird items are called columns

2. Grid Rows

The horizontal lines of grid items are called rows.

3. Grid Gaps

The space between each column/row are called gaps.

4. Grid Lines

  • The lines between columns are called column lines.
  • The lines between rows are called row lines

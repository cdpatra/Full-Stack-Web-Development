Position

There are mainly 4 types of postion

  1. static (default) -normal flow
  2. relative
  3. fixed
  4. absolute
  5. sticky

There are even sub-properties

  1. top
  2. bottom
  3. left
  4. right
    5. but these 4 sub-properties are only applicable for relative, fixed or absolute not for static

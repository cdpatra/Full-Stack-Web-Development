Specificity

  • Specificity tells if one element is styled by more than one rule, then which style will be applicable
  • More specific rule will get more value

General Rule

  • !important will override everything
  • Inline CSS can override everything except !important
  • External will work according to specificity

Specificity Rule

  • (#) id selector has highest value
  • (*) Universal selector has lowest value

Specificity Rule

Trick (Point System)

  • 10,000 point to !important
  • 1000 to Inline
  • 100 to id
  • 10 to class,attribute or pseudo-class
  • 1 for element selector & pseudo-elements
  • 0 to universal selector
    • and at last add all the points then compare the specificity
NOTE:- If same rule written two or more times in external sheet then last rule will be applicable

Selectors in CSS

1. Type Selector

example: 

        H1{
            color:red;
        }

2. ID Selector

example: 

        #red-p{
            color:red;
        }

3. Class Selector

example: 

        .red-p{
            color:red;
        }

4. Pseudo Class and Pseudo Element

example: 

        a:hover{
            color:red;
        }
        p::first-line{
            color:red;
        }

5. Attribute Selector

example: 

        a[href="https://example.com"]{
            color:red;
        }

6. Universal Selector

example: 

        *{
            color:red;
        }

7. Combinator Selector

Descendant combinatory (child at any stage)

example: 

        .red-p span{
            color:red;
        }

Child combinatory (Direct Child)

example: 

        .red-p>span{
            color:red;
        }

Adjacent combinatory (right after sibling)

example: 

        .red-p + h1{
            color:red;
        }

General sibling combinatory (all sibling)

example: 

        .red-p~h1{
            color:red;
        }

8. Add Selector

example: 

        .red-p, h1{
            color:red;
        }
